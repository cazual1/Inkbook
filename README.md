# InkBook 🖋️

**Tattoo Artist Discovery & Booking Platform**

A mobile-first web platform where tattoo artists showcase their work, manage appointments, offer payment plans, and do FaceTime consultations. Users discover artists near them, browse portfolios, read reviews, and book sessions.

## Features

- **Artist Discovery** — Browse by style (Black & Grey, Neo-Traditional, Japanese, etc.)
- **Portfolio Showcase** — Full gallery with lightbox view
- **Booking System** — Date/time picker with availability
- **Payment Plans** — Pay in full, split, 4 payments, or monthly
- **FaceTime Consults** — Video consultation booking
- **Community Feed** — Share ink, healing updates, recommendations
- **Reviews & Ratings** — Verified client reviews
- **Artist Profiles** — Dedicated pages with full portfolio

## Tech

Single-page React app. No build step needed — runs with Babel standalone.
